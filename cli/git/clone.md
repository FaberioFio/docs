## Clone

##### Clone, then save to a folder

```
git clone --depth=1 https://github.com/Bash-it/bash-it.git bashit
```

##### Depth 1

```
git clone --depth=1 https://github.com/Bash-it/bash-it.git
```

##### Copy one file's content to another file

```
curl https://raw.githubusercontent.com/avelino/awesome-go/master/README.md -o index.md
```
